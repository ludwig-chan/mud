# 文字冒险经营游戏

这是一个基于 Vue 3 + TypeScript 开发的文字冒险经营类游戏。游戏采用回合制（每回合代表游戏中的一天），玩家需要为每一天规划行动。游戏特意设计为"慢节奏"：游戏中的一天相当于现实世界的5分钟，在这段时间内，玩家的角色会完成指定的行动。这种设计既能让玩家感受到游戏进程，又能避免过度沉迷。

## 时间系统

- 游戏时间与现实时间的关系：
  - 游戏中的1天 = 现实世界的5分钟
  - 游戏中的1月 = 现实世界的2.5小时
  - 游戏中的1年 = 现实世界的30小时

- 回合制玩法：
  - 每个回合代表游戏中的一天
  - 玩家在每天开始时规划当天的行动
  - 行动开始后需要等待现实时间流逝
  - 等待期间可以离开游戏，系统会自动完成行动

- 日常行动示例：
  - 砍树：消耗1天时间（5分钟）
  - 采集果实：消耗1天时间（5分钟）
  - 耕地：消耗1天时间（5分钟）
  - 建造小型设施：消耗2-3天时间（10-15分钟）
  - 建造大型建筑：消耗7-15天时间（35-75分钟）

## 游戏玩法说明

### 日常行动系统
- 每天只能选择一个主要行动：
  - 砍树：消耗1天（5分钟），获得10个木材
  - 采集野果：消耗1天（5分钟），获得8-12个野果
  - 挖矿：消耗1天（5分钟），获得5-8个矿石
  - 狩猎：消耗1天（5分钟），获得3-5个兽肉
  - 钓鱼：消耗1天（5分钟），获得4-6条鱼

### 种植系统
- 种植操作：
  - 种植树苗：消耗1天（5分钟）
  - 树苗成长：30天（2.5小时）
  - 果树结果：60天（5小时）
- 农作物生长周期：
  - 小麦：15天（75分钟）
  - 水稻：20天（100分钟）
  - 蔬菜：10天（50分钟）
  - 药草：25天（125分钟）

### 战斗系统
- 角色初始生命值为100
- 通过装备提升攻击力和防御力
- 战斗采用回合制机制

### 金币系统
- 初始金币：100金
- 通过出售资源赚取金币
  - 木材：5金/个
  - 野果：10金/个
  - 矿石：20金/个
- 完成任务获得金币奖励
- 金币用途：
  - 购买装备和道具
  - 招募队伍伙伴
  - 升级设施

### 队伍系统
- 最多可招募3名伙伴
- 伙伴类型：
  - 战士：擅长近战
  - 射手：擅长远程攻击
  - 医师：提供治疗支持
  - 工匠：加快采集速度
- 伙伴招募费用：1000-5000金币不等
- 伙伴每日维护费：50金币/人

### 建筑系统
- 基础建筑：
  - 农舍：提供居住空间
  - 仓库：储存资源
  - 畜棚：饲养牲畜
  - 谷仓：储存粮食
- 生产建筑：
  - 磨坊：加工粮食
  - 工坊：制作工具
  - 铁匠铺：打造装备
- 军事建筑：
  - 兵营：训练士兵
  - 箭塔：防御设施
  - 城墙：保护村庄

### 牲畜系统
- 可饲养的牲畜：
  - 猪：产出肉食
  - 牛：可用于耕地
  - 羊：产出羊毛
- 饲养方式：
  - 圈养：消耗饲料，安全系数高
  - 放养：无需饲料，每日损失率5%
- 饲料需求：
  - 猪：10单位/天
  - 牛：20单位/天
  - 羊：8单位/天
- 劳动力价值：
  - 牛：相当于3个工人的耕地效率
  - 羊：可帮助收割牧草

### 生命系统
- 角色属性：
  - 年龄：从18岁开始
  - 健康状况：会随年龄下降
  - 疾病风险：随年龄增加
  - 预期寿命：60-80岁
- 死亡原因：
  - 自然衰老
  - 战斗伤亡
  - 疾病感染
  - 意外事故

### 家族系统
- 婚姻关系：
  - 20岁后可以结婚
  - 需要与NPC建立感情
  - 婚后可以生育子女
- 子女培养：
  - 每个子女有独特天赋
  - 可以培养不同技能
  - 选择继承人培养
- 家族传承：
  - 主角死亡后选择继承人
  - 继承家族资产与地位
  - 获得家族声望加成

### 科技系统
- 农业科技：
  - 高产作物培育
  - 机械化耕作
  - 现代化温室
- 工业科技：
  - 蒸汽动力
  - 电力系统
  - 自动化生产线
- 军事科技：
  - 冷兵器锻造
  - 火药武器研发
  - 现代化军备
  - 太空武器系统
- 医疗科技：
  - 基础医疗
  - 现代医院
  - 生命延长技术
- 太空科技：
  - 航天器研发
  - 太空站建设
  - 行星开发
  - 太空作战系统

### 疾病系统
- 疾病类型：
  - 普通疾病：感冒、发烧
  - 慢性疾病：随年龄增长
  - 传染病：需要隔离
  - 职业病：与工作相关
- 预防措施：
  - 建立医疗设施
  - 研发医疗技术
  - 注意作息调养
  - 定期体检
